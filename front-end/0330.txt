- 개발환경을 시작할떄는 npm start를 사용하지만
   빌드를 할떄는 npm run build를 사용한다..

npm start을 사용하면 불필요한 용량이있는데
npm run build를 사용하면 실제 제품에서 사용하도록 용량을 줄이던지 보안을 처리해서 사용가능하도록 한다..

즉 실제 서비스 할때는 build안에 파일을 사용.
